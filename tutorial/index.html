Instalasi Offline tanpa video

Ini adalah tata cara instalasi aplikasi web KPK Digital Flipbook tanpa video

Hai saya kali ini akan memandu kamu dalam proses instalasi aplikasi web KPK Digital Flipbook. Sebelum kita mulai pastikan kamu telah menyiapkan aplikasi yang di butuhkan di bawah ini.

  • Browser Versi terbaru Unduh Firefox
  • Web Server Unduh XAMPP

Untuk web browser pastikan memilih yang sesuain dengan OS yang kamu pakai. Disini saya menggunakan OS Windows 64 bit maka saya akan mendownload versi windows yang terbaru

Jika sudah selesai mendownload ikuti panduan install xampp pada gambar di bawah ini

Pada gambar di atas pastikan bahwa folder KPK_Digital_Flipbooks sudah di pindahkan pada folder C:\xampp\htdocs
